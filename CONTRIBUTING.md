# Contributing to SolAgent

## Setup

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Validate changes: `npm test`

## Contribution Guidelines

1. Keep docs aligned with actual implementation.
2. Add or update tests/checks when changing behavior.
3. Prefer small, focused pull requests with clear descriptions.
4. Do not commit private keys, RPC secrets, or local wallet files.

## Project Scope

This repository is currently a challenge submission prototype. If you add
features, clearly mark whether they are mock/demo logic or production-ready.
